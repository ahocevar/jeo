A lightweight spatial library for the JVM

jeo is a library for building spatial applications that provides tools for geometry and projection support in addition to drivers for a number of spatial formats. The goal of jeo is provide simple and convenient apis for developers that require spatial functions in their applications. Jeo is licensed under the Apache Version 2.0 license.
